# 命令的基本格式

## 命令的提示符

```
[root@localhost ~]# 
```

* []：这是提示符的分隔符号，没有特殊含义。
* root：显示的是当前的登录用户。
* @：分隔符号，没有特殊含义。
* localhost：当前系统的简写主机名（完整主机名为localhost.domain）。
* ～：代表用户当前所在的目录，此例中用户当前所在的目录是家目录。
* #：命令提示符。超级用户是#，普通用户是$

## 命令的基本格式

```
[root@localhost ~]#  命令 [选项] [参数]
```
* 选项：是用于调整命令的功能
* 参数：是命令的操作对象，如果省略参数，是因为有默认参数

## 绝对路径和相对路径

* 绝对路径：以根目录为参照物，从根目录开始，一级一级进入目录
* 相对路径：以当前目录作为参照物，进行目录查找
